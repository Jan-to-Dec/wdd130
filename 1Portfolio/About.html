Data analysts store data in a database, which is then analyzed by a data engineer (or backend developer) Data engineers create and manage dashboards for data that needs to be monitored or for specific data.

Data engineers collect and process data from apps or the web and store it in a way that is easy for data analysts to use. Develop dashboards for data analysts to use The current trend is for data engineers to either 1) do backend development, 2) do machine learning/deep learning, or 3) focus on data engineering.

Machine learning engineers develop the necessary models in their domain (Computer Vision, Natural Language Process, etc.). They apply the developed models to production (actual products). Monitor the performance of the model and improve the performance of the model or create a better model The key here is to put it into production.

Data scientists usually work in R&D organizations and do thesis research. If you work on the data analytics side, you do statistical modeling. They do what machine learning engineers do (develop models, improve model performance, develop additional models)
